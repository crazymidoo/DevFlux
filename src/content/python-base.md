## Capitolo 1: Introduzione a Python

Python non è solo un linguaggio di programmazione; è una filosofia di design che mette al primo posto la **leggibilità** e la **semplicità**. Creato da Guido van Rossum nel 1991, oggi è il linguaggio più popolare al mondo per il Machine Learning, l'Analisi Dati e l'Automazione.

### Perché scegliere Python?

* **Sintassi Pulita:** Si legge quasi come l'inglese, riducendo la "paura del codice".
* **Versatilità:** Puoi creare siti web, analizzare dati o automatizzare compiti ripetitivi.
* **Batteries Included:** Python arriva con una vastissima libreria standard che ti permette di fare quasi tutto subito.

### L'ambiente di lavoro

Prima di scrivere codice, devi assicurarti che Python sia installato sul tuo computer.

1.  **Download:** Scarica l'ultima versione da [python.org](https://www.python.org/).
2.  **Verifica:** Apri il terminale (o prompt dei comandi) e digita:
    ```bash
    python --version
    ```
3.  **Editor:** Ti consiglio di usare **Visual Studio Code** per scrivere i tuoi file `.py`. Assicurati di installare l'estensione "Python" ufficiale.
